# MultiRTSPViewer
> python + pyqt5 + opencv를 이용한 다중 rtsp 뷰어입니다.

PyQt5와 OpenCV를 이용하여 4개의 RTSP 스트림을 보여 줄 수 있는 다중 RTSP 스트림 뷰어입니다.
RTSP 스트림을 보여 줄 때, Thread를 사용합니다.

## 개발 & 실행 환경

Python3 필요

PyQt5, qdarkstyle, imutils, opencv-python 모듈 필요(pip로 설치하시면 됩니다.)
